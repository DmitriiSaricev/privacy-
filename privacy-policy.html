Политика конфиденциальности

Последнее обновление: 12 марта 2026 г.

1. Общие положения

Настоящая политика конфиденциальности описывает, как Dmitrii Agency («мы», «нас») собирает, использует и защищает информацию пользователей приложения Wiza Bot, взаимодействующих с нами через Facebook Messenger и Instagram.

2. Какие данные мы собираем

В процессе работы бота мы можем собирать следующие данные:

  • Идентификатор пользователя Facebook или Instagram (User ID)
  • Текст сообщений, направленных боту
  • Имя пользователя (публичное имя профиля)
  • Дата и время обращения

3. Цели использования данных

Собранные данные используются исключительно для:

  • Ответа на запросы пользователей в рамках консультации по мебельной продукции
  • Организации замера и записи на встречу
  • Улучшения качества обслуживания

4. Хранение данных

История переписки хранится в защищённой базе данных и используется только для обеспечения контекста разговора. Мы не продаём и не передаём ваши данные третьим лицам в коммерческих целях.

5. Данные Meta Platforms

Приложение использует Facebook Messenger API и Instagram API в соответствии с условиями платформы Meta. Мы запрашиваем только минимально необходимые разрешения для работы чат-бота. Ваши данные обрабатываются в соответствии с Политикой конфиденциальности Meta.

6. Права пользователей

Вы имеете право:

  • Запросить удаление ваших данных
  • Получить информацию о хранящихся данных
  • Отказаться от дальнейшей обработки

Для этого напишите нам на контактный email ниже.
